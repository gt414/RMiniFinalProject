# 103~107年性別毒品罪入監曲線圖
[![RLang](https://img.shields.io/badge/Language-R-brown.svg)](https://www.r-project.org)

這次的mini project主要是統計103到107年因毒品罪入監的男生及女生的曲線圖。其中的內容包含罪名別、教育程度、性別、日期及統計值。
